# Skin-Uri リリースノート

## Version 0.5.0

_**Webサイト名**: Skin-Uri(ゲーム内スキン販売サイト)
_**リリース日**: 2025-12-5
_**リリースバージョン**: 0.5.0

### 概要
Version 0.5.0では、商品データベースを構築し、ゲーム別のカテゴリ管理や一括上新機能を実装。
また、トップページと商品ページを新たに追加し、基本的な商品閲覧が可能になりました。

### 目的
- 商品管理システムの基盤構築を目的としています。

### リリース内容 / 新機能
- 商品データベースを新規構築  
  - 商品情報を一元管理できる仕組みを実装  
  - ゲームごとのカテゴリ分けに対応  


- 基本的なサイト構造を構築  
  - トップページの初期デザインを追加  
  - 商品一覧ページを実装し、商品をサイト上で閲覧可能に 


### 今後の開発予定
商品検索・フィルター機能、およびカート機能の追加を予定しています。
