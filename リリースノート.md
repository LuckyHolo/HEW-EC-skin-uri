# Skin-Uri リリースノート

## Version 0.5.0

- **Webサイト名**: Skin-Uri(ゲーム内スキン販売サイト)
- **リリース日**: 2025-12-5
- **リリースバージョン**: 0.5.0

### 概要
- Version 0.5.0では、商品データベースを構築し、ゲーム別のカテゴリ管理や一括上新機能を実装。
- また、トップページと商品ページを新たに追加し、基本的な商品閲覧が可能になりました。

### 目的
- 商品管理システムの基盤構築を目的としています。

### リリース内容 / 新機能
- 商品データベースを新規構築  
  - 商品情報を一元管理できる仕組みを実装  
  - ゲームごとのカテゴリ分けに対応  


- 基本的なサイト構造を構築  
  - トップページの初期デザインを追加  
  - 商品一覧ページを実装し、商品をサイト上で閲覧可能に 


### 今後の開発予定
商品検索・フィルター機能、およびカート機能の追加を予定しています。

## Version 1.5.0
## **リリース日:2026-02-13**
### 修正内容
- カート上限を5個までに制限をかけました。
  (新機能の*¹クーポンが使用できるようになったため)*¹詳しくは新機能のクーポンで  
  カート上限に達すると通知でお知らせするようになっています。
- アカウントページにて総購入額表示をするようになりました。
- 一度購入したアイテムを再度購入できないようにしました。
  カートに入れるボタンが購入済みになります。
- 重複アイテムを不可にしました。
  同じ商品をカートに入れられないようにしました。
- マイコレクションタブ
  - マイページに購入済みスキン一覧を表示させるようにしました。
  - 価格別にアイテムをグループ化して表示（Legendary / Epic/ Standard）
- 軽微な修正をしました。
  - クーポンタブのUI修正
  - トースト通知で操作結果を即時フィードバック


### 新機能
- クーポン機能を追加しました
  購入額に応じてクーポンを付与します。
  - ¥5,000ごと → 5%クーポン
  - ¥20,000ごと → 10%クーポン
  - ¥50,000達成 → 50%クーポン（1回のみ）
  クーポンが%クーポンなので多額減額にならないようにカートを制限しました。
