# Skin-Uri リリースノート

## Version 0.5.0

_**Webサイト名**: Skin-Uri (ゲーム内スキン販売サイト)_  
_**リリース日**: 2025-12-5_  
_**リリースバージョン**: 0.5.0_

### 概要
Version 0.5.0では、Skin Uri ショップサイトの機能強化と UI 改善を行いました。  
Ajax による検索・フィルター機能を追加し、チャンピオンごとのスキン表示もスムーズになりました。

### 目的
- 商品検索やフィルターの利便性向上  
- UI/UX の改善  
- Ajax を用いた操作のスムーズ化  

### リリース内容 / 新機能
- 商品一覧の Ajax 検索・フィルター  
  - キーワード検索  
  - カテゴリー別フィルター  
  - 価格帯指定  

- チャンピオンごとのスキン絞り込み  

- ページネーション Ajax 対応  
  - 次ページクリック時もフィルター状態を維持  

- 商品カードに詳細ページ・カート追加ボタン  

- お気に入り機能 (Toggle Favourite) Ajax 対応  

- チャンピオンを追加: **10名**  
- スキンを追加: **118個**

### 改善点 / Improvements
- UI/UX の改善  
  - サイドバー・チャンピオンリストのスクロールバーをカスタム  
  - レスポンシブデザイン対応  
  - トースト通知で操作フィードバック  
  - スキン画像のホバー時アニメーション  

### バグ修正 / Bug Fixes
- ページネーションクリック時にチャンピオンフィルターがリセットされる問題を修正  
- カート追加・お気に入りボタンの反応改善  

### 備考 / Notes
- 今後は注文完了後のメール送信機能を実装予定  
- Django バージョンと依存ライブラリは requirements.txt に記載  

### 実行方法 / How to Run
1. 仮想環境を作成・有効化  
   python -m venv venv
   source venv/bin/activate  # Mac/Linux
   venv\Scripts\activate     # Windows

2. pip install -r requirements.txt

3. python manage.py runserver

4. ブラウザでアクセス　http://127.0.0.1:8000/
